VK Messenger

A simple VK messenger for desktop on .NET platform.

Download this project as a .zip file Download this project as a tar.gz file

Описание

VK Messenger — простой текстовый мессенджер для социальной сети ВКонтакте (VK).

Функции

  • Просмотр всех диалогов (включая чаты)
  • Загрузка последних 10 сообщений в выбранном диалоге
  • Мгновенное получение и отображение новых сообщений с помощью системного уведомления
  • Новые сообщения отображаются в истории открытого диалога
  • Для каждого сообщения отображается полное имя, аватар автора и время отправки сообщения
  • Возможность минимизировать приложение в трей
